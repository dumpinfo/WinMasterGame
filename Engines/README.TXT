
README.TXT for ENGINES directory

Within this directory are a number of commerical 2D/3D engines
for your use and evaluation. 

GENESIS3D   <DIR> - The Genesis 3D engine by Eclipse Software and on-line books.
POWERRENDER <DIR> - The PowerRender engine by Chris Egerter of Egerter Software.

To load and install any of the engines simply change directories and
read the README.TXT files for specific information. In most cases, you just
need to launch a self extracting .EXE file and that will load each engine
respectively.
